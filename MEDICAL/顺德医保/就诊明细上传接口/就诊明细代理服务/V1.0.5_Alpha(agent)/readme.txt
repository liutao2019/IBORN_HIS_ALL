 代理服务器(Agent)
【功能描述】
      实现请求数据转发，对请求数据进行压缩/解压，数据指纹验证，加密等，并记录请求和接收到的数据日志。
　　
【使用说明】
　　1.设置参数
　　		参数统一在agnet.xml文件中设置
	【初始参数】		
		端口(port)					设置代理本地监听端口	如：9090
		线程池大小(threadPoolSize)	通常设置端口数大1/3，如有10个终端，则设置为13	
		服务器接口地址				proxyGroup > proxy > url 属性值 。
									通常地址是http://<本机IP地址>:<设置的端口号>/<应用上下文名>/services/AgentService?wsdl
			          				如：http://127.0.0.1:8080/web/services/AgentService?wsdl
			          				
	【日志参数】	日志节点：<logger>	
		日志级别	日志级别,DEBUG,INFO,ERROR	
		日志路径	节点：dirPath	
	　　
2.启动代理
	 2.1 首先要确定已安装好JDK1.7，并且在命令窗口中输入 java -version 显示的是jdk1.7版本。
	 2.2 双击代服务器目录下的"启动.bat"文件。启动成功后，控制台会提示服务地址，否则启动失败。
		服务地址通常是：http://<本机IP地址>:<设置的端口号>/facadeService?wsdl

 医疗调用 DLL（可选）
【功能描述】
	用于加密数据的函数。被加密的数据不能超过16字符长，不允许为空串（即被加密的数据的长度应大于0小于或等于16字节），返回的密文为16位或32位长的16进制字符串，比如“897689F2D8D17422DD959D060A6E8A26”。
【功能定义】
        int encrypt(const char* plaintext,char* ciphertext)

【参数说明】
	const char* plaintext：要加密的字符，被加密的数据长度>0，且<=16字节，不允许为空串。
	char* ciphertext：被加密的数据，返回的密文为16位或32位长的16进制字符串。
【返回值】
	  成功返回0；否则表示执行失败。
